ROUTES:

NEW USER:

This route is authentication-free, it is allowed to create a new user without any type of authentication

JSON format for User Creation:

POST: { "firstName": " ", "lastName": " ", "email": " ", "phone": " ", "roles": " ", "password": " " },

USERS:

JSON format for Users:

GET: { "_id": " ", "firstName": " ", "lastName": " ", "email": " ", "phone": " ", "roles": " ", "password": " ", },

PATH: { "id": " ", "firstName": " ", "lastName": " ", "email": " ", "phone": " ", "roles": " ", "password": " " },

DELETE: { "id": " ", },
